# 2〜36進数 計算・変換アプリ (Base Calculator & Converter)

TypeScript で実装された、2進数から36進数までの相互変換および四則演算ができるシンプルな Web アプリケーションです。

## 🌟 特徴

- **広範な進数サポート**: コンピュータが標準で扱える 2進数〜36進数（0-9, A-Z）のすべてに対応しています。
- **リアルタイムバリデーション**: 選択した進数に対して不正な文字（例: 2進数における「2」や、16進数における「G」など）が入力された場合、即座にエラーを検知して画面に通知します。
- **進数別の四則演算**: 任意の進数のまま、足し算・引き算・掛け算・割り算（商）の計算が可能です。

## 🛠 搭載機能

1. **① 進数変換 (Converter)**
   - 指定した進数の数値を、別の任意の進数へ一発で変換します。
2. **② 進数計算 (Calculator)**
   - 同じ進数同士の数値を使って、その進数のまま四則演算を行います。

## 🚀 動かし方

ローカル環境で動かすには以下の手順を行います。

### 1. 環境準備
事前に Node.js および TypeScript がインストールされていることを確認してください。

### 2. コンパイル
TypeScript ファイル (`app.ts`) を JavaScript (`app.js`) にコンパイルします。
```bash
tsc app.ts --target es6
3. アプリの起動
コンパイルが終わると app.js が生成されます。
その後は、index.html を直接ブラウザで開くか、VS Code の拡張機能 Live Server などを用いて起動してください。
